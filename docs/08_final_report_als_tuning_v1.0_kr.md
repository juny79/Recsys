# ALS 하이퍼파라미터 튜닝 최종 보고서

**문서 버전:** v1.0  
**작성일:** 2026-02-08  
**작성자:** Antigravity  
**관련 태스크:** ALS 모델 성능 최적화

---

## 1. 개요 (Executive Summary)

본 프로젝트는 커머스 구매 예측 모델의 NDCG@10 성능 향상을 목표로 진행되었습니다. 베이스라인 ALS 모델(NDCG 0.0955)에서 출발하여, SASRec 모델 도입 시도와 실패 분석을 거쳐 다시 ALS 모델의 체계적인 하이퍼파라미터 튜닝을 수행했습니다.

**최종 결과:**
- **Best NDCG@10:** **0.1124** (베이스라인 대비 **+17.7%** 향상)
- **최종 모델:** ALS (Alternating Least Squares)
- **핵심 요인:** 잠재 요인(Factors) 증대, 정규화(Regularization) 최적화, 그리고 **강력한 시간 감쇠(Time Decay)** 적용

---

## 2. 튜닝 프로세스 및 결과

총 5단계(Phase)의 실험을 통해 점진적으로 성능을 향상시켰습니다.

### 📊 단계별 성능 변화 요약

| 단계 | 튜닝 대상 | 최적 설정 | NDCG@10 | 변화율 | 비고 |
|:---:|:---|:---|:---|:---|:---|
| **Baseline** | 초기 모델 | factors=32 | 0.0955 | - | 기준점 |
| **Phase 1** | Factors | **128** | 0.1091 | +14.2% | 표현력 증대 효과 큼 |
| **Phase 2** | Regularization | **0.1** | 0.1099 | +15.1% | 과적합 방지 미세 효과 |
| **Phase 3** | Alpha | **10** (기본값) | 0.1099 | - | 고신뢰도(40, 100)는 성능 하락 |
| **Phase 4** | Event Weights | **1/3/5** (기본값) | 0.1099 | - | 구매 희소성으로 인해 가중치 증대 효과 미미 |
| **Phase 5** | Decay Rate | **0.1** | **0.1124** | **+17.7%** | **최신 트렌드 반영 중요** |

## 3. 상세 분석

### 3.1 Factors (잠재 요인)
- 32 → 64 → 128로 늘릴수록 성능이 비례하여 증가했습니다.
- 사용자-아이템 간의 복잡한 상호작용 패턴을 학습하기 위해서는 충분한 차원의 임베딩 벡터가 필요함을 확인했습니다.

### 3.2 Regularization (정규화)
- 0.001(기본값) → 0.01 → 0.1로 높였을 때 소폭의 성능 향상이 있었습니다.
- 데이터 희소성으로 인한 노이즈를 억제하는 데 도움이 되었습니다.

### 3.3 Alpha (선호도 신뢰도)
- 기본값 10이 최적이었으며, 값을 높일수록(40, 100) 성능이 저하되었습니다.
- 암시적 피드백(Implicit Feedback) 데이터에서 관측된 상호작용에 지나친 확신을 갖는 것은 오히려 해가 됨을 시사합니다.

### 3.4 Event Weights (이벤트 가중치)
- `view=1, cart=3, purchase=5` (기본 설정)가 가장 좋았습니다.
- 구매 데이터가 전체의 0.02%로 매우 희소하여, 구매 가중치를 10, 20으로 높여도 전체 추천 품질에 긍정적인 영향을 주기 어려웠습니다. 오히려 노이즈로 작용한 것으로 보입니다.

### 3.5 Time Decay (시간 감쇠)
- **결정적 요인:** `decay_rate=0.1` (반감기 약 7일)에서 가장 높은 성능을 기록했습니다.
- 이는 커머스 데이터 특성상 **사용자의 최신 관심사가 구매로 어질 가능성이 매우 높음**을 의미합니다. 오래된 이력보다 최근 일주일 내의 행동에 집중하는 것이 추천 정확도를 높이는 열쇠였습니다.

## 4. 결론 및 제언

### ✅ 최종 추천 모델 설정 (Best Benchmark)
다음 설정을 프로덕션 모델의 기본값으로 사용할 것을 권장합니다.
- `factors`: **128**
- `regularization`: **0.1**
- `alpha`: **10**
- `decay_rate`: **0.1** (강력한 최신성 반영)
- `event_weights`: **{'view': 1.0, 'cart': 3.0, 'purchase': 5.0}**

### 🚀 향후 발전 방향
1. **SASRec 재시도 (장기 과제):** 현재 데이터 부족(Short Sequence)으로 ALS에 패배했지만, 데이터가 더 축적된다면 시퀀스 모델이 다시 유효할 수 있습니다.
2. **하이브리드 모델:** ALS의 협업 필터링 결과와 최신 인기도(Popularity)를 결합하거나, LightGCN 같은 최신 그래프 기반 모델 도입을 고려해볼 수 있습니다.
3. **실시간성 강화:** `decay_rate`의 중요성이 확인된 만큼, 실시간 세션 기반 추천(Session-based RecSys) 도입도 유효할 것입니다.
