## 🔍 NDCG 계산 원리 간단 설명

### NDCG (Normalized Discounted Cumulative Gain)

**핵심 개념:**
- **Cumulative Gain (CG)**: 추천된 상품의 관련성 합계
- **Discounted**: 순위가 낮을수록 가중치 감소 (상위권이 중요)
- **Normalized**: 이상적인 추천과 비교하여 0-1로 정규화

**공식:**
```
NDCG@10 = DCG@10 / IDCG@10

DCG@10 = Σ(rel_i / log2(i+1))  (i=1 to 10)
```

**예시:**
- 1위 추천이 관련 있으면: 높은 점수 (log2(2) = 1로 나눔)
- 10위 추천이 관련 있으면: 낮은 점수 (log2(11) ≈ 3.46으로 나눔)

→ **상위권 정확도가 점수에 가장 큰 영향**

   - 10개 추천 중 평균 **4-5개가 사용자 관심사 적중**

   - SASRec, 앙상블 등으로 0.12-0.15 달성 가능

1. **순위 정확도**
