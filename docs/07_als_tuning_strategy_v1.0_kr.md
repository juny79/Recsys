# ALS 하이퍼파라미터 튜닝 전략

## 📊 현재 상태

### 현재 성능
| 설정 | NDCG@10 |
|------|---------|
| 기본 ALS + 이벤트 가중치 + 시간 감쇠 | **0.0955** |

### 현재 하이퍼파라미터 (train_als.py)
```python
num_factor = 32           # 잠재 요인 수
regularization = 0.001    # L2 정규화
alpha = 10                # 신뢰도 스케일링
```

### 현재 가중치 설정
```python
event_weights = {'view': 1.0, 'cart': 3.0, 'purchase': 5.0}
decay_rate = 0.03
```

---

## 🎯 튜닝 전략

### 1단계: 잠재 요인 수 (factors) 조정

> [!IMPORTANT]
> 가장 영향력 있는 파라미터. 모델 용량을 결정합니다.

| 설정 | 값 | 기대 효과 |
|------|-----|----------|
| 현재 | 32 | 기준선 |
| 테스트 1 | **64** | 더 복잡한 패턴 학습 |
| 테스트 2 | **128** | 고용량 모델 |
| 테스트 3 | 16 | 과적합 방지 (비교용) |

**권장**: 64 또는 128 시도 (아이템 29,502개에 32는 다소 작음)

---

### 2단계: 정규화 (regularization) 조정

> 과적합을 방지하는 L2 제약

| 설정 | 값 | 기대 효과 |
|------|-----|----------|
| 현재 | 0.001 | 기준선 (약한 정규화) |
| 테스트 1 | **0.01** | 중간 정규화 |
| 테스트 2 | **0.1** | 강한 정규화 |
| 테스트 3 | 0.0001 | 더 약한 정규화 |

**권장**: factors 증가 시 regularization도 함께 증가 (0.01 ~ 0.1)

---

### 3단계: Alpha (신뢰도 스케일링) 조정

> 암묵적 피드백의 신뢰도를 조절

| 설정 | 값 | 기대 효과 |
|------|-----|----------|
| 현재 | 10 | 기준선 |
| 테스트 1 | **40** | 높은 가중치 아이템 강조 |
| 테스트 2 | **100** | 구매 이벤트 극강조 |
| 테스트 3 | 1 | 가중치 차이 축소 |

**권장**: 40-100 범위 테스트 (구매 이벤트 강조)

---

### 4단계: 이벤트 가중치 조정

> 구매 → 장바구니 → 조회 순으로 중요도 반영

| 설정 | view | cart | purchase | 의미 |
|------|------|------|----------|------|
| 현재 | 1.0 | 3.0 | 5.0 | 기준선 |
| 테스트 1 | 1.0 | 5.0 | **10.0** | 구매 강조 |
| 테스트 2 | 0.5 | 3.0 | **20.0** | 구매 극강조 |
| 테스트 3 | 1.0 | 2.0 | 3.0 | 차이 축소 |

**권장**: 구매 가중치를 10-20으로 높이기

---

### 5단계: 시간 감쇠율 조정

| 설정 | decay_rate | 반감기 | 기대 효과 |
|------|------------|--------|----------|
| 현재 | 0.03 | ~23일 | 기준선 |
| 테스트 1 | **0.01** | ~69일 | 오래된 데이터도 중요 |
| 테스트 2 | **0.05** | ~14일 | 최신 데이터 강조 |
| 테스트 3 | 0.1 | ~7일 | 극최신 강조 |

## 🔬 권장 실험 순서

### Phase 1: 핵심 파라미터 (factors)
```bash
# 실험 1: factors=64
python train_als.py --num_factor 64

# 실험 2: factors=128
python train_als.py --num_factor 128
```

### Phase 2: 최적 factors + regularization
```bash
# 최적 factors에서 regularization 조정
python train_als.py --num_factor 64 --regularization 0.01
python train_als.py --num_factor 64 --regularization 0.1
```

### Phase 3: Alpha 조정
```bash
python train_als.py --num_factor 64 --regularization 0.01 --alpha 40
python train_als.py --num_factor 64 --regularization 0.01 --alpha 100
```
