## 2. 4-Way 성공 요인 분석

### 2.1 모델별 기여도 추정

| 모델 | 단독 성능 | 4-Way 가중치 | 추정 기여도 | 역할 |
|---|:---:|:---:|:---:|---|
| **ALS** | 0.1124 | 50% | 최대 | 정적 취향 (CF) |
| **SASRec** | 0.0864 | 25% | 중간 | 시퀀스 패턴 |
| **XGBoost** | 0.1117 | 10% | 소량 | 인기도 + LTR |
| **CatBoost** | 0.1125 | 15% | 중간 | LTR + 궤적 피처 |

### 2.2 3-Way vs 4-Way 비교

| 지표 | 3-Way | 4-Way | 차이 |
|---|:---:|:---:|:---:|
| **Base Score** | 0.1196? | 0.1196? | - |
| **부스트 후** | 0.1326 | 0.1341 | **+1.13%** |
| **부스트 효율** | 10.9% | **12.1%** | **+1.2%p** |
| **후보 풀** | 23.9개/유저 | 25.8개/유저 | +8.0% |
| **모델 수** | 3개 | 4개 | +33% |

**핵심 발견:**
- 4-Way의 부스트 효율이 **12.1%**로 3-Way(10.9%)보다 높음
- CatBoost 추가로 후보 다양성 증가 → Phase 1 부스트가 더 효과적으로 작동
- 더 넓은 후보 풀(25.8개)에서 진짜 Top-10을 더 잘 선별

### 2.3 왜 3-Way보다 4-Way가 더 좋은가?

**1. 더 넓은 후보 커버리지**
```
2-Way: 유저당 ~17개 후보
3-Way: 유저당 23.9개 후보 (+40%)
4-Way: 유저당 25.8개 후보 (+52%)
```

**2. 상호보완적 시그널**
- ALS: 전체 사용자 패턴 (CF)
- SASRec: 최근 시퀀스 패턴
- XGBoost: 인기도 + 랭킹 시그널
- **CatBoost: 궤적 피처 + 복잡한 상호작용**

**3. 노이즈 상쇄 효과**
- 각 모델이 10-25% 가중치로 분산
- 한 모델의 오류가 전체에 미치는 영향 감소
- "Wisdom of Crowds" 효과

**4. Phase 1 부스트와의 시너지**
- 더 많은 후보 중에서 진짜 Top 아이템 발굴
- User history (purchase, cart, recency)와 잘 맞는 아이템을 더 효과적으로 부스트

## 3. 추가 최적화 방향

### 3.1 가중치 최적화 ⭐⭐⭐⭐⭐

**현재 4-Way 가중치:**
- ALS=0.5, SASRec=0.25, XGB=0.1, CatBoost=0.15

**최적화 후보:**

#### Option A: ALS 강화 (보수적)
- **ALS=0.55**, SASRec=0.25, XGB=0.08, CatBoost=0.12
- 근거: ALS(0.1124)가 가장 강력한 단일 모델
- 예상: 0.1342 - 0.1345

#### Option B: CatBoost 강화 (공격적)
- ALS=0.48, SASRec=0.25, XGB=0.07, **CatBoost=0.20**
- 근거: CatBoost(0.1125)가 XGB보다 높고, LTR 피처가 Phase 1과 시너지
- 예상: 0.1343 - 0.1348

#### Option C: 균형 조정
- ALS=0.52, SASRec=0.26, XGB=0.10, CatBoost=0.12
- 근거: SASRec 약간 증가, CatBoost 약간 감소로 안정성 확보
- 예상: 0.1341 - 0.1344

---

### 3.2 4-Way + Phase 1+ Enhanced ⭐⭐⭐⭐⭐

**전략:** 4-Way (현재 가중치) + 더 강한 부스트

**Phase 1 → Phase 1++:**
- Repeat: 1.3/2.0/2.5 → **1.5/2.2/2.7**
- Event: 1.8x/1.4x → **2.0x/1.6x**
- Recency: 2.0x/1.5x/1.2x → **2.2x/1.7x/1.4x**
- Top-3: 1.5x → **1.7x**

**근거:**
- 4-Way에서 부스트 효율이 2-Way 수준(12.1%)으로 회복
- 더 강한 부스트 적용 여유 생김
- Phase 3의 과도한 부스트(24.64배)보다는 훨씬 보수적 (최대 13배)

**예상:** 0.1345 - 0.1355

**리스크:** Phase 3 재현 가능성 있지만, 4-Way base가 더 견고해서 안전

---

### 3.3 스코어링 방법 최적화 ⭐⭐⭐⭐

**현재:** RRF (Reciprocal Rank Fusion), K=60

**대안 1: K값 조정**
- K=40: 상위 랭크에 더 높은 가중치 → 정확도 중시
- K=90: 하위 후보도 고려 → 다양성 중시
- 예상: K=40이 nDCG에 유리할 수 있음

**대안 2: rank_inv 방식**
- 공식: score = 1 / rank (RRF는 1/(K+rank))
- 더 공격적으로 상위 랭크 우선
- 예상: 0.1342 - 0.1350

**대안 3: Weighted RRF**
- 각 모델의 단독 성능에 따라 K값 다르게 적용
- ALS, CatBoost → K=40 (더 신뢰)
- SASRec → K=80 (덜 신뢰)

## 5. 실험 결과 종합 분석

### 5.1 가중치 실험 교훈

| 실험 | 가중치 변화 | 결과 | 교훈 |
|---|---|:---:|---|
| 3-Way base | ALS=0.6 | 0.1326 | 기준점 |
| 3-Way w70 | ALS=0.7 → XGB=0.05 | 0.1320 ❌ | XGB 과소 = 실패 |
| **4-Way** | **+CatBoost=0.15** | **0.1341** ✅ | **다양성 = 성공** |

**결론:**
- 단순히 강한 모델(ALS) 비중 증가 ≠ 성능 향상
- 다양한 모델의 상호보완 > 한 모델 과의존
- XGB(0.1117), CatBoost(0.1125) 같은 "준수한" 모델도 중요한 기여

### 5.2 부스트 효율 추이

| 앙상블 | Base | 부스트 후 | 효율 |
|---|:---:|:---:|:---:|
| 2-Way | 0.1173 | 0.1330 | **13.4%** |
| 3-Way | 0.1196 | 0.1326 | 10.9% |
| **4-Way** | **0.1196** | **0.1341** | **12.1%** |

**발견:**
- 모델 수 증가 → 부스트 효율 회복
- 더 넓은 후보 풀 → Phase 1이 더 잘 작동
- 4-Way의 12.1% 효율은 2-Way의 13.4%에 근접
