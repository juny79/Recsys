# EDA Insight Report: NDCG@10 향상을 위한 핵심 시그널

## 1. 분석 결과 요약

### 🔍 시그널 1: 유저 로열티 (Repeat Rate) - **18.3%**
- **현상:** 유저가 이전에 상호작용했던 아이템을 다시 볼 확률이 18.3%에 달합니다. 
- **인사이트:** 추천 목록 상단에 '유저가 이미 봤던 아이템' 중 다시 볼 가능성이 높은 아이템을 배치하는 것만으로도 NDCG를 크게 올릴 수 있습니다. (기존 모델들은 새로운 아이템 발굴에 집중하여 이 부분을 놓쳤을 가능성이 큼)

### 🔍 시그널 2: 카테고리 집착 (Category Consistency) - **68.4%**
- **현상:** 직전 행동과 동일한 카테고리의 아이템을 다음 행동으로 선택할 확률이 매우 높습니다.
- **인사이트:** 유저의 '마지막 카테고리'에 해당하는 아이템 가중치를 대폭 상향해야 합니다. 특히 세션 내(짧은 시간 간격) 활동에서는 이 경향이 절대적입니다.

### 🔍 시그널 3: 세션 폭발성 (Temporal Bursts)
- **현상:** 상호작용 간 시간 간격의 중앙값(Median)이 **0.8분(48초)**입니다. 반면 평균은 20시간입니다.
- **인사이트:** 유저는 한 번 들어오면 짧은 시간 동안 집중적으로 쇼핑을 합니다. 이 '폭발적인 세션' 기간 동안은 앞서 언급한 카테고리 일관성과 로열티가 극대화되므로, 리얼타임성 피처가 점수 경쟁력의 핵심입니다.

## 2. 권장 전략: [Loyalty-Boosted Ensemble]

단순히 모델 점수를 합치는 대신, 다음의 'Post-Processing' 규칙을 적용합니다.

1.  **Re-view Boost:** 유저의 이력에 존재하는 아이템이 추천 후보(Top-100)에 포함되어 있다면, 해당 아이템의 랭킹을 최상단으로 보정합니다. (특히 2회 이상 발생한 경우)
2.  **Category Filter:** 상위 10개 아이템 선정 시, 유저의 '최근 3개 카테고리'에 해당하는 아이템이 적어도 50% 이상 포함되도록 강제합니다.
3.  **Fine-grained Decay:** 현재 1일 단위 Decay보다 더 촘촘한(예: 1시간 단위) Decay를 적용하여 '방금 전의 행동'에 더 큰 가중치를 부여합니다.

## 3. 결론
지금까지의 LTR(머신러닝) 모델이 실패한 이유는, 데이터 전체를 평균적으로 보려고 했기 때문입니다. 위 분석 기반의 **'데이터 기반 휴리스틱 보정'**을 앙상블 결과에 적용한다면 0.12의 벽을 충분히 넘을 수 있을 것으로 보입니다.
