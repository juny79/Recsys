# nDCG@10 추가 점수 상승 전략

**현재 달성:** 0.1265 (Repeat Boost만 적용)  
**목표:** 0.13~0.14 (약 3~10% 추가 향상)

---

## 🎯 단계별 전략 (우선순위 순)

### **전략 1: Event Type 차등 Boost** ⭐⭐⭐⭐⭐
**예상 효과:** +3~5% (0.1265 → 0.1303~0.1328)  
**구현 난이도:** 낮음  
**실행 시간:** 2분

#### 핵심 아이디어
- 현재: 모든 재방문을 동일하게 취급
- 개선: **purchase/cart 이력**이 있는 재방문 아이템에 더 큰 가중치

#### 적용 방법
```python
# 현재: count만 고려
if count >= 3: boost = 2.5
elif count >= 2: boost = 2.0

# 개선: event_type도 고려
if 'purchase' in event_types:
    boost *= 1.8  # 구매 이력 → 1.8배 추가
elif 'cart' in event_types:
    boost *= 1.4  # 장바구니 이력 → 1.4배 추가

# 예: 3회 봤고 그중 구매 1번 → 2.5 * 1.8 = 4.5배!
```

#### 왜 효과적인가?
- **구매 의도가 가장 강한 시그널**
- nDCG는 상위 3개가 60% 차지 → 구매 이력 아이템을 최상단에 배치

---

### **전략 2: Category Consistency (최근 세션)** ⭐⭐⭐⭐
**예상 효과:** +2~4% (누적 0.1303 → 0.1329~0.1355)  
**구현 난이도:** 중간  
**실행 시간:** 3분

#### 핵심 아이디어
- 68.4% 카테고리 일관성 활용
- 유저의 **최근 2~3개 카테고리**에 속한 아이템 우선

#### 적용 방법
```python
# 최근 카테고리 매칭 부스트
if item_category in user_recent_categories[:2]:
    boost *= 1.5  # 최근 2개 카테고리
elif item_category in user_recent_categories[:4]:
    boost *= 1.2  # 최근 4개 카테고리
```

#### 구현 포인트
- **재방문 아이템 중에서도** 최근 카테고리 우선
- 조합 예: 재방문(2.5x) × 최근카테고리(1.5x) = 3.75배

---

### **전략 3: Recency Window (시간 감쇠)** ⭐⭐⭐⭐
**예상 효과:** +1~3% (누적 0.1329 → 0.1342~0.1369)  
**구현 난이도:** 중간  
**실행 시간:** 3분

#### 핵심 아이디어
- 중앙값 48초 (세션 내 집중 쇼핑) 활용
- **마지막 상호작용으로부터의 시간** 고려

#### 적용 방법
```python
# 마지막 상호작용 후 경과 시간 계산
hours_since = (user_last_time - item_last_time).hours

if hours_since <= 1:
    recency_boost = 2.0  # 1시간 이내
elif hours_since <= 24:
    recency_boost = 1.5  # 1일 이내
elif hours_since <= 72:
    recency_boost = 1.2  # 3일 이내
else:
    recency_boost = 1.0
```

#### 조합 효과
- 재방문(2.5x) × 구매이력(1.8x) × 최근(2.0x) = **9배!**
- 오래된 재방문보다 **최근 재방문**이 훨씬 강력한 시그널

---

### **전략 4: Top-3 집중 최적화** ⭐⭐⭐⭐⭐
**예상 효과:** +2~5% (누적 0.1342 → 0.1369~0.1409)  
**구현 난이도:** 낮음  
**실행 시간:** 1분

#### 핵심 아이디어
- nDCG에서 **1~3위가 전체 점수의 60% 차지**
- Top-3에 가장 확실한 아이템만 배치

#### 적용 방법
```python
# Top-3 강제 배치 규칙
# 1. 구매 이력이 있는 재방문 아이템
# 2. 최근 1일 이내 + 재방문
# 3. 3회 이상 재방문

top3_candidates = []
for item in candidates:
    if has_purchase and repeat_count >= 2:
        score += 10.0  # 매우 강한 부스트
        top3_candidates.append(item)

# 상위 3개는 확실한 것만
final_top10 = top3_candidates[:3] + other_candidates[:7]
```

---

### **전략 5: Diversity Penalty (하위권)** ⭐⭐⭐
**예상 효과:** +1~2% (누적 0.1369 → 0.1383~0.1409)  
**구현 난이도:** 낮음  
**실행 시간:** 1분

#### 핵심 아이디어
- Top-3는 정확도 최우선
- 4~10위는 다양성 확보 (혹시 모를 관심사)

#### 적용 방법
```python
# 4~10위에 카테고리 다양성 적용
if position >= 4:
    if category_count[item_category] >= 3:
        score *= 0.8  # 같은 카테고리 3개 이상이면 감점
```

## 💡 핵심 인사이트

1. **구매 이력 = 최강 시그널**
   - purchase가 있는 재방문은 거의 확실한 관심사

2. **상위 3개가 전부다**
   - Top-3에 확실한 것만, 나머지는 다양성

3. **최근성 × 재방문 = 황금 조합**
   - 오래된 3회보다 최근 2회가 더 강력

4. **점진적 적용**
   - 한 번에 모든 전략 적용 시 오버피팅 가능
   - Phase별로 검증하며 적용
