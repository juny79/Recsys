## 2. 핵심 인사이트

### 2.1 XGB의 역할 재평가
**이전 가설 (틀림):**
- XGB(0.1117)가 노이즈를 더한다
- ALS 비중을 높이면 2-Way(0.1330)에 근접할 것

**실험 결과 (사실):**
- XGB 기여도를 0.15 → 0.05로 줄이니 오히려 **0.45% 악화**
- XGB는 ALS/SASRec이 놓친 **popularity + LTR 시그널을 제공**
- 0.6/0.25/0.15 가중치가 이미 **near-optimal**

### 2.2 가중치 vs 부스트
```
문제 진단:
- 3-Way(0.1326)가 2-Way(0.1330)보다 낮은 이유
- ❌ 가중치 문제가 아님 (w70 실험으로 확인)
- ✅ 부스트 효율 문제 (13.4% → 10.9%)
```

**결론:** 가중치는 그대로 두고 **부스트 강화**가 답

## 4. 실패한 전략 분석

### ❌ 가중치 조정 (ALS 증가)
**시도:** 3-Way w70 (ALS=0.70, XGB=0.05)
**결과:** 0.1320 (-0.45%)
**교훈:**
1. **XGB는 노이즈가 아니라 시그널이었음**
2. **2-Way(0.7/0.3)의 성공을 3-Way에 단순 적용 불가**
3. **모델이 추가되면 최적 가중치도 변함**

### ❌ 2-Way 가중치 모방 실패 이유
```
2-Way (0.7 ALS / 0.3 SASRec):
→ SASRec(0.0864)이 약해서 30%만 줌
→ ALS(0.1124)가 월등히 강해서 70% 필요

3-Way (0.6 ALS / 0.25 SASRec / 0.15 XGB):
→ XGB(0.1117)가 ALS와 비슷한 수준
→ 3개 모델이 균형을 이루려면 ALS 60%가 적정
→ ALS 70%로 올리면 XGB 시그널 손실
```

## 5. 수학적 분석

### 5.1 부스트 효율 비교

| 앙상블 | Base | 부스트 후 | 효율 | 효율 감소 |
|---|:---:|:---:|:---:|:---:|
| 2-Way | 0.1173 | 0.1330 | 13.4% | - |
| 3-Way (0.6/0.25/0.15) | 0.1196 | 0.1326 | 10.9% | **-2.5%p** |
| 3-Way (0.7/0.25/0.05) | 0.1196 | 0.1320 | 10.4% | **-3.0%p** |

### 5.2 Phase 1+ 기대 효과

만약 부스트 효율을 13.4% → 12.0%로 회복한다면:
- 0.1196 × 1.120 = **0.1340**

Phase 1+의 부스트 강화:
- 평균 부스트 증가율: ~7%
- 기대 효율: 10.9% × 1.07 ≈ 11.7%
- 예상 점수: 0.1196 × 1.117 ≈ **0.1336**

**보수적 예상:** 0.1330 - 0.1335
