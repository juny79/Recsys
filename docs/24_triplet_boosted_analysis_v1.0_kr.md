# 3-Way Ensemble + Phase 1 Boost 전략 분석

## 1. Phase 7 결과로부터의 교훈

### Phase 7 Conservative 실험 결과
- **설정:** Phase 1보다 약한 부스트 (Repeat: 1.2/1.7/2.1, Event: 1.5x/1.25x, Recency: 1.6x/1.3x)
- **결과:** 0.1289 (Phase 1 대비 -3.1%)
- **교훈:** **부스트 강도를 줄이면 오히려 성능 악화**

### 최적 부스트 강도 검증
Phase 1의 부스트 값들은 이미 최적에 가까운 것으로 확인됨:
- Repeat Boost: 1.3 → 2.0 → 2.5 (방문 횟수별)
- Event Multiplier: Purchase 1.8x, Cart 1.4x
- Recency Multiplier: ≤1h: 2.0x, ≤24h: 1.5x, ≤72h: 1.2x
- Top-3 Boost: 1.5x

## 2. 3-Way Ensemble + Phase 1 전략

### 2.1 Base Ensemble 비교

| 앙상블 | 구성 | Base Score | Phase 1 적용 | 최종 Score |
|:---|:---|:---:|:---:|:---:|
| **2-Way** | ALS + SASRec | 0.1173 | ✅ | **0.1330** (+13.4%) |
| **3-Way** | ALS + SASRec + XGB | 0.1196 | ❌ | 0.1196 |
| **3-Way + Phase 1** | ALS + SASRec + XGB | 0.1196 | ✅ | **??? (미제출)** |

### 2.2 3-Way 앙상블의 장점
1. **더 넓은 후보 풀**
   - 2-Way: 유저당 평균 ~17개 후보
   - 3-Way: 유저당 평균 **23.9개 후보** (+40%)
   
2. **상호보완적 시그널**
   - **ALS**: 정적 취향 (Collaborative Filtering)
   - **SASRec**: 시퀀스 패턴 (Sequential)
   - **XGBoost**: 인기도 + 랭킹 시그널 (Popularity + LTR)

3. **검증된 Base Score**
   - 3-Way는 이미 0.1196으로 2-Way(0.1173)보다 높음
   - XGBoost 단독도 0.1117로 준수한 성능

### 2.3 예상 성능

**보수적 예상 (같은 부스트 효과율):**
- 2-Way에서 Phase 1이 +13.4% 향상
- 3-Way(0.1196) × 1.134 = **0.1356**

**현실적 예상 범위:**
- **Best Case:** 0.1350 - 0.1360 (XGB 시너지 최대)
- **Expected:** 0.1340 - 0.1350 (부스트 효과 약간 감소)
- **Worst Case:** 0.1330 - 0.1340 (2-Way와 비슷)

### 2.4 왜 더 높을 수 있는가?

1. **더 정확한 Base Ranking**
   - 3-Way base가 이미 +2.0% 더 높음
   - Phase 1 boost가 더 정확한 후보에 적용됨

2. **XGB의 Popularity 시그널**
   - 최근 인기 아이템을 Phase 1 boost로 더 끌어올림
   - Recency boost + Popularity가 시너지

3. **후보 다양성**
   - ALS/SASRec이 놓친 후보를 XGB가 복구
   - 부스트가 더 넓은 범위에서 작동

## 3. 리스크 분석

### 3.1 하방 리스크
- **Noise 증가:** XGB가 잘못된 후보 추가 → Phase 1이 잘못된 아이템 부스트
- **과적합 우려:** XGB LTR은 Label Bias 가능성
- **가중치 불균형:** 3모델 가중치(0.6/0.25/0.15)가 최적이 아닐 수 있음

### 3.2 상방 리스크
- **시너지 효과:** 3모델 상호보완 + Phase 1 boost가 예상 이상 작동
- **Top-3 Boost 극대화:** 더 넓은 후보 풀에서 진짜 Top-3 발굴

## 6. 결론

Phase 7의 0.1289는 "약한 부스트는 안 된다"는 것을 확인시켜줬습니다. 
이제 남은 것은 **"더 좋은 Base + 검증된 강한 부스트"** 조합입니다.

3-Way + Phase 1은:
- ✅ 검증된 Base (0.1196)
- ✅ 검증된 Boost (Phase 1)
- ✅ 상호보완적 시그널 (ALS+SASRec+XGB)
- ❌ 유일한 미검증 요소: 이 조합

**수학적 기대값**: 0.1196 × 1.134 = **0.1356**
**보수적 예상**: **0.1340 - 0.1350**

지금까지의 모든 실험 중 **가장 논리적이고 기대값이 높은 전략**입니다.
